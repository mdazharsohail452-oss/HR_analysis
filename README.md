# HR Analytics: Employee Attrition Study
📌 Project Overview
Image:
![image alt](http://github.com/mdazharsohail452-oss/HR_analysis/blob/main/powerBi%20image.png)
Employee attrition is a critical challenge for organizations as it affects productivity, increases recruitment costs, and impacts workforce stability. This project focuses on analyzing employee attrition using data-driven techniques and interactive visualizations. The dashboard helps HR teams understand attrition patterns and make informed decisions to improve employee retention.

🎯 Objectives
To analyze employee attrition trends using HR data
To identify key factors influencing employee turnover
To provide interactive visual insights for HR decision-making
To support data-driven employee retention strategies

📂 Dataset
Dataset Name: IBM HR Analytics Employee Attrition Dataset
Source: Kaggle
Records: 1,470 employees
Target Variable: Attrition (Yes / No)

🛠 Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Microsoft Excel (for data preprocessing)

🔍 Methodology
Data collection and understanding
Data cleaning and preprocessing
Feature engineering (Age Group, Attrition Rate)
Exploratory data analysis
Dashboard design and visualization
Insight generation and interpretation

📊 Dashboard Features
KPI cards for total employees, attrition count, and attrition rate
Attrition analysis by department, job role, age group, and gender
Salary, experience, and satisfaction-based attrition insights
Interactive filters and slicers for dynamic exploration

📈 Results

The dashboard effectively identifies high-risk attrition segments based on demographic and job-related factors. Insights show higher attrition among younger employees, lower-income groups, and employees with low satisfaction levels. The visual analytics approach supports accurate and reliable HR decision-making.

✅ Conclusion

The HR Employee Attrition Analysis Dashboard provides a comprehensive and interactive solution for understanding employee turnover. It enables HR teams to identify critical attrition drivers and take proactive measures to improve employee retention.
